# Nuxt 学习仓库 Agent 说明

这个仓库是一个 Nuxt 学习项目，面向已经会 Vue、正在学习 Nuxt、SSR、企业后台和 AI 应用落地的前端开发者。

## 每次开始前必须读取

每次 Agent 进入这个项目，先读取：

```text
docs/ai-learning-checklist.md
```

读取后需要先确认：

- 当前学习阶段进行到哪里。
- 哪些任务已经完成。
- 下一步应该继续做什么。

不要跳过学习清单直接开始改代码。

## 项目定位

- 技术栈：Nuxt 4、Vue 3、TypeScript、Vite、Element Plus
- 学习方式：边看代码边跟敲，而不是只看文档
- 当前方向：企业工单后台 + AI 工单草稿 + 后端接口 + 后续数据库/通知/移动端
- 目标：示例尽量小、清晰、可运行，适合学习、讲解和求职作品展示

## 协作规则

- 优先保留“适合学习”的代码结构，不要为了抽象而抽象。
- 修改尽量围绕当前学习主题，不做无关重构。
- 写注释时，重点解释 Nuxt、SSR、后端接口、AI 边界，不重复基础 Vue 语法。
- 处理权限时，明确区分“页面 middleware 拦截”和“服务端真实鉴权”。
- 每完成一个学习任务后，更新 `docs/ai-learning-checklist.md` 的勾选状态和学习记录。
