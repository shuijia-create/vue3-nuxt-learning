# Nuxt 跟敲学习目录

你已经会 Vue，所以这里按“前端转 Nuxt”的路线设计，不重复讲 Vue 基础。

## 先跟敲这 3 天

1. [Day 1：Vue 开发者上手 Nuxt 约定](./day-01.md)
2. [Day 2：Nuxt 数据获取、Server API 与登录会话](./day-02-login.md)
3. [Day 3：Nuxt Middleware、页面权限与服务端权限边界](./day-03-permission.md)

## 你要重点掌握

- `pages/` 文件路由
- `layouts/` 布局系统
- `server/api/` 服务端接口
- `useFetch`、`useAsyncData`、`$fetch`
- `useState` 跨组件状态
- `middleware/` 路由权限
- SSR、CSR、SSG 的区别
- cookie 登录态在 SSR 下如何处理

## 启动命令

```bash
npm run dev
```

访问：

```text
http://localhost:3000
```
